🌍 AIR QUALITY MEASURING SYSTEM

Real-Time Air Monitoring Using PM2.5 Sensors and IoT Technology

📖 Project Overview

This project develops a low-cost air quality monitoring system capable of measuring PM2.5 concentration, temperature and humidity in real time. Data are processed by a microcontroller and displayed on an LCD screen while being stored online using ThingSpeak.

⚙️ Main Components

PM2.5 Sensor

Measures particulate matter concentration in the air.

Temperature Sensor

Monitors environmental temperature.

Humidity Sensor

Measures relative humidity.

Microcontroller

Processes sensor data and controls the system.

LCD Display

Displays measurements in real time.

ThingSpeak

Cloud platform for data visualization and storage.

🔄 System Workflow

  1. Sensors collect environmental data.
  2. Data are sent to the microcontroller.
  3. The microcontroller processes the readings.
  4. Results are displayed on LCD.
  5. Data are uploaded to ThingSpeak for monitoring.
